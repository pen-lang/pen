# Command line tools

The `pen` command has the following sub-commands.

## `init` command

### Initializing an application package

```sh
pen init foo
```

### Initializing a library package

```sh
pen init --lib foo
```

## `build` command

The `build` sub-command builds a package of a current directory.

```sh
pen build
```
